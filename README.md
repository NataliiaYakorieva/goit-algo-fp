# Висновки щодо моделювання кидків двох кубиків методом Монте-Карло

У ході виконання симуляції методом Монте-Карло було здійснено 1 000 000 кидків двох гральних кубиків. Для кожного кидка обчислювалася сума чисел, що випали на кубиках, та підраховувалася кількість появ кожної можливої суми.

Отримані ймовірності для кожної суми були порівняні з аналітичними розрахунками, які базуються на теорії ймовірностей (кількість сприятливих комбінацій поділена на загальну кількість можливих комбінацій — 36).

**Результати симуляції показали, що ймовірності, отримані методом Монте-Карло, дуже близькі до аналітичних значень.** Відхилення незначні й зменшуються зі збільшенням кількості симуляцій, що підтверджує правильність реалізації алгоритму та ефективність методу Монте-Карло для оцінки ймовірностей у подібних задачах.

Графік і таблиця, побудовані за результатами симуляції, наочно демонструють збіг експериментальних і теоретичних ймовірностей для кожної суми від 2 до 12.

**Висновок:**  
Метод Монте-Карло дозволяє з високою точністю оцінити ймовірності випадання сум при киданні двох кубиків, і результати симуляції підтверджують аналітичні розрахунки.
